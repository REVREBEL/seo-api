# postgresql-17-plus

A provider-neutral Agent Skill for PostgreSQL 17+ schema design, performance tuning, operations, reliability, backup/recovery, and code review.

## Structure

```text
postgresql-17-plus/
├── SKILL.md
└── references/
    ├── schema/
    ├── performance/
    ├── operations/
    ├── development/
    └── security/
```

Install by copying the `postgresql-17-plus` directory into your agent skills directory, such as `.agents/skills/` or the equivalent directory for your agent client.

The master `SKILL.md` stays intentionally concise and loads the detailed reference files on demand.
